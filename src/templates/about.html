Plug in your stack

Sensor integrations are ready today.

Post sensor data from ESP32, Raspberry Pi, Home Assistant, or any cloud pipeline. We store any schema with a timestamp and source.

Endpoint: /api/ingest

Auth: X-API-Key header

Fields: use data to send any shape

Schema registry: /api/schemas

curl -X POST http://localhost:5000/api/ingest \
  -H "Content-Type: application/json" \
  -H "X-API-Key: dev-sensor-token" \
  -d '{
    "device_id": "gh-1-main",
    "greenhouse_id": 1,
    "schema": "climate_v1",
    "data": {
      "temperature_c": 26.4,
      "humidity_pct": 67.2,
      "light_lux": 740,
      "co2_ppm": 520
    }
  }'
