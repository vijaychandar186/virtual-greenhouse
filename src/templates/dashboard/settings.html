API Key

Use this key to push sensor data to /api/ingest. Send it as the X-API-Key header or ?api_key= query param. Data ingested with this key is automatically linked to your account.

Example — push a reading:

curl -X POST {{ request.host_url }}api/ingest \
  -H "X-API-Key: {{ user.api_key }}" \
  -H "Content-Type: application/json" \
  -d '{"device_id":"my-sensor","schema":"climate_v1","data":{"temperature_c":24.5,"humidity_pct":65}}'
